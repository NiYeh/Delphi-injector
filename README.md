## 使用方式

**選擇你要注入的軟件和要注入的dll文件**

**確保注入器已顯示成功獲取要注入的軟件名稱和dll文件路徑**

![1](https://github.com/user-attachments/assets/c55faf77-ec94-48a9-be10-65d824a79a4e)

**按下"注入"即可**
